## Description

Feature flags [Nest](https://github.com/nestjs/nest) MVP.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Client

### Dev

```bash
cd client

npm ci

npm run dev
```

### Build

```bash
cd client

npm ci

npm run build
```
